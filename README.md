Vi har skapat ett API som är baserat på python och flask, där man kan söka hotell, kontrollera kapacitet och hantera bokningar. Skapa använder samt logga in.
Syftet med denna applikation är i utbildningssyfte.

Tekniker vi används oss av... 
Backend Flask (python 3.10, Pycharm) 
Databas (postgreSQL)
Databas verktyg (SQLAlchemy) + våra text queris
Sessionshantering (Flasksessioner)
API typ (RESTful) JSON API

Installation & körning...
Krav (Python 3.10)
Pycharm
PostgreSQL (installerat & tillgängligt)
Git & Github
Nummer 1 Tryck på denna länk för att komma till vårat github repository https://github.com/LinneaSoderman/Matalskarens_odysse.git
Nummer 2 Installera databas dump i repositorys så du har den lokalt
Nummer 3 Klona SSH nyckeln i pycharm 
Nummer 4 Installera dependencies genom denna kod, pip install -r -requirements.txt
Nummer 5 Starta applikation genom att trycka run
Nummer 6 API startar normalt på http://127.0.0.1:5000/

Konfiguration... 
Skapa en .env fil där du kopierar in .env.example och ändrar texten till din personliga databas server 
Se till så att .gitignore finns annars behövs det installeras.
